## How to Install

### Step 1: Install Tampermonkey
Download and install the **Tampermonkey** extension for your browser:
- [Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- [Safari (Mac App Store)](https://apps.apple.com/app/tampermonkey/id1459806926)
- [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/tampermonkey/)

### Step 2: Install Tui-Qiao
Click here: **[🚀 Install Script](https://github.com/small-thinking/tui-qiao/raw/main/first-principle-auditor.user.js)**. Tampermonkey will automatically detect the script and prompt you to install.

### Step 3: Configure API Key
1. Select any text on any webpage.
2. Click the **🔍 Tui-Qiao** button.
3. Click the ⚙️ icon in the panel and paste your **Gemini API Key**.

## Features
- **Standalone:** No backend required.
- **Categorized Audit:** Proactive evidence seeking for rumors vs logic analysis for opinions.
- **Privacy First:** Your API key is stored locally in your browser.
