# Tui-Qiao (推敲) 🧠

**Truth Seeking through Logical Rigor.**

[中文说明 (README_ZH.md)](./README_ZH.md) | **[🚀 Install Userscript (一键安装)](https://github.com/small-thinking/tui-qiao/raw/main/first-principle-auditor.user.js)**

Tui-Qiao (推敲) is a standalone, selection-based auditing tool. It acts as a cognitive filter to deconstruct claims, identify hidden assumptions, and uncover first principles directly in your browser.

### Features
- **Standalone:** No backend required. Calls Gemini API directly.
- **One-Click Installation:** Managed via Tampermonkey.
- **Privacy First:** Your API key is stored locally in your browser.

### Quick Start
1. **Install:** Click the **[Install Link](https://github.com/small-thinking/tui-qiao/raw/main/first-principle-auditor.user.js)** (Requires Tampermonkey).
2. **Configure:** Click the ⚙️ icon in the Tui-Qiao panel to enter your Gemini API Key.
3. **Audit:** Select any text and click the 🧠 button.
