# Tui-Qiao (推敲) 🧠

**Truth Seeking through Logical Rigor.**

[中文说明 (README_ZH.md)](./README_ZH.md)

Tui-Qiao (推敲) is a standalone, selection-based auditing tool. It acts as a cognitive filter to deconstruct claims, identify hidden assumptions, and uncover first principles directly in your browser.

### Features
- **Standalone:** No backend required. Calls Gemini API directly.
- **Hidden Assumptions:** Uncover what the author is taking for granted.
- **Logical Leaps:** Identify gaps between premises and conclusions.
- **Cui Bono:** Determine who benefits from the propagation of this message.
- **First Principles:** Reconstruct core truths from the ground up.

### Quick Start
1. **Install Userscript:** Copy the content of `first-principle-auditor.user.js` and install it in your browser extension (e.g., Tampermonkey).
2. **Configure:** Click the ⚙️ icon in the Tui-Qiao panel to enter your Gemini API Key.
3. **Audit:** Select any text and click the 🧠 button.
