# Tui-Qiao (推敲) 🧠

**Truth Seeking through Logical Rigor.**

Tui-Qiao (推敲) is a minimalist, selection-based auditing tool designed to help you deconstruct claims, identify hidden assumptions, and uncover the first principles behind any message.

In an era of information overload and "vibes-based" decision making, Tui-Qiao acts as a cognitive filter, stripping away emotional adjectives to find the underlying logic (or lack thereof).

### Features
- **Hidden Assumptions:** Uncover what the author is taking for granted.
- **Logical Leaps:** Identify gaps between premises and conclusions.
- **Cui Bono:** Determine who benefits from the propagation of this message.
- **First Principles:** Reconstruct the core truths from the ground up.

### Quick Start
1. **Run Local Bridge (Mac/Linux)**
   ```bash
   git clone https://github.com/small-thinking/tui-qiao.git
   cd tui-qiao
   npm install express cors
   node bridge.js
   ```
2. **Install Userscript**
   Copy the content of `first-principle-auditor.user.js` and install it in your browser extension (e.g., Tampermonkey).
