## 安装指南

### 第一步：安装 Tampermonkey
首先为你的浏览器安装 **Tampermonkey（油猴）** 插件：
- [Chrome 商店](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- [Safari (Mac App Store)](https://apps.apple.com/app/tampermonkey/id1459806926)
- [Firefox 扩展](https://addons.mozilla.org/firefox/addon/tampermonkey/)

### 第二步：安装“推敲”脚本
点击链接：**[🚀 安装 Tui-Qiao 脚本](https://github.com/small-thinking/tui-qiao/raw/main/first-principle-auditor.user.js)**。油猴插件会自动识别并弹出安装界面。

### 第三步：配置 API Key
1. 在任意网页划选一段文字。
2. 点击出现的 **🔍 Tui-Qiao** 按钮。
3. 点击面板上的 ⚙️ 图标，填入你的 **Gemini API Key** 即可。

## 核心功能
- **完全独立：** 无需本地服务器，直连 Google Gemini API。
- **三级审计：** 智能识别个人感言（忽略）、传闻（查证）与观点（逻辑分析）。
- **隐私安全：** 你的 API Key 仅保存在浏览器本地。
