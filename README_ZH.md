# 推敲 (Tui-Qiao) 🧠

**求真，从逻辑严密性开始。**

“推敲”是一个极简的划词审计工具，旨在帮助你拆解各类主张、识别隐藏假设，并挖掘任何信息背后的第一性原理。

在信息过载和“氛围化”决策的时代，“推敲”充当了认知过滤器的角色，剥离感性的修饰语，寻找底层的逻辑。

### 核心功能
- **隐藏假设 (Hidden Assumptions):** 挖掘作者视为理所当然的前提。
- **逻辑断层 (Logical Leaps):** 识别前提与结论之间的脱节。
- **利益相关 (Cui Bono):** 确定谁从这条信息的传播中获益。
- **第一性原理重构 (First Principles):** 从零开始重构核心事实。

### 快速开始
1. **运行本地 Bridge (Mac/Linux)**
   ```bash
   git clone https://github.com/small-thinking/tui-qiao.git
   cd tui-qiao
   npm install express cors
   node gemini_bridge.js
   ```
2. **安装油猴脚本**
   将 `first-principle-auditor.user.js` 的内容安装到 Tampermonkey 中。
