# 推敲 (Tui-Qiao) 🧠

**求真，从逻辑严密性开始。**

[English Description (README.md)](./README.md)

“推敲”是一个独立的划词审计工具，无需后端支持。它在浏览器中充当认知过滤器，帮助你拆解各类主张、识别隐藏假设，并挖掘底层的第一性原理。

### 核心功能
- **独立运行：** 无需本地服务器，直连 Gemini API。
- **隐藏假设 (Hidden Assumptions):** 挖掘作者视为理所当然的前提。
- **逻辑断层 (Logical Leaps):** 识别前提与结论之间的脱节。
- **利益相关 (Cui Bono):** 确定谁从这条信息的传播中获益。
- **第一性原理重构 (First Principles):** 从零开始重构核心事实。

### 快速开始
1. **安装油猴脚本：** 将 `first-principle-auditor.user.js` 的内容安装到 Tampermonkey 中。
2. **配置：** 点击面板上的 ⚙️ 图标，填入你的 Gemini API Key。
3. **开始审计：** 划选网页上的任何文字，点击出现的 🧠 按钮。
