# 推敲 (Tui-Qiao) 🧠

**求真，从逻辑严密性开始。**

[English Description (README.md)](./README.md) | **[🚀 一键安装脚本](https://github.com/small-thinking/tui-qiao/raw/main/first-principle-auditor.user.js)**

“推敲”是一个独立的划词审计工具，无需后端支持。它在浏览器中充当认知过滤器，帮助你拆解各类主张、识别隐藏假设，并挖掘底层的第一性原理。

### 核心功能
- **独立运行：** 无需本地服务器，直连 Gemini API。
- **安装便捷：** 支持 Tampermonkey 一键安装与自动更新。
- **隐私安全：** API Key 仅保存在你的浏览器本地。

### 快速开始
1. **安装：** 点击 **[一键安装链接](https://github.com/small-thinking/tui-qiao/raw/main/first-principle-auditor.user.js)**（需安装 Tampermonkey 插件）。
2. **配置：** 点击面板上的 ⚙️ 图标，填入你的 Gemini API Key。
3. **开始审计：** 划选网页上的任何文字，点击出现的 🧠 按钮。
